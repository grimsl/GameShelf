A modern web application for tracking and sharing your gaming library, built with React, TypeScript, and AWS Amplify Gen 2.

## Features

- **User Authentication**: Sign up and sign in with email
- **Game Library Management**: Add, edit, and organize your games
- **Status Tracking**: Track games as playing, finished, paused, dropped, or planning
- **Rating System**: Rate games from 1-5 stars
- **Notes & Reviews**: Add personal notes and reviews for each game
- **Public Profiles**: Share your gaming library with others
- **Responsive Design**: Clean, gaming-focused UI that works on all devices

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Backend**: AWS Amplify Gen 2
- **Authentication**: AWS Cognito
- **Database**: AWS AppSync (GraphQL)
- **Styling**: Custom CSS with gaming-focused dark theme
- **Icons**: Lucide React
- **Notifications**: React Hot Toast

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn
- AWS CLI configured (for deployment)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd GameShelf
```

2. Install dependencies:
```bash
npm install
```

3. Set up AWS Amplify:
```bash
npm install -g @aws-amplify/cli
amplify configure
```

4. Deploy the backend:
```bash
npm run amplify:dev
```

5. Start the development server:
```bash
npm run dev
```

### Environment Variables

Copy `env.example` to `.env` and fill in the values generated by Amplify:

```bash
cp env.example .env
```

The environment variables will be automatically populated when you run `npm run amplify:dev`.

## Data Models

### User
- `id`: Unique identifier
- `userHandle`: Unique username
- `email`: User's email address
- `avatarUrl`: Profile picture URL
- `bio`: User biography

### Game
- `id`: Unique identifier
- `title`: Game title
- `cover`: Game cover image URL
- `description`: Game description

### LibraryEntry
- `id`: Unique identifier
- `userId`: Reference to user
- `gameId`: Reference to game
- `status`: playing, finished, paused, dropped, planning
- `rating`: 1-5 star rating
- `notes`: Personal notes/review
- `gameTitle`: Denormalized game title
- `gameCover`: Denormalized game cover
- `createdAt`: Entry creation date
- `updatedAt`: Last update date
