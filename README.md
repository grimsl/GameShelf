GameShelf

GS is a modern web application for tracking and sharing your gaming library. It was built with React, TypeScript, and AWS Amplify Gen 2.

Main Features: 

- User Authentication: Sign up and sign in with email
- Game Library Management: Add, edit, and organize your games
- Status Tracking: Track games as playing, finished, paused, dropped, or planning
- Rating System: Rate games from 1-5 stars
- Notes & Reviews: Add personal notes and reviews for each game
- Public Profiles: Share your gaming library with others
- Friends: Add friends to share your games with others, and vice versa
- Responsive Design: Clean, gaming-focused UI that works on all devices

The current Tech Stack

- Frontend: React 19, TypeScript, Vite
- Backend: AWS Amplify Gen 2
- Authentication: AWS Cognito
- Database: AWS AppSync (GraphQL)
- Styling: Custom CSS with gaming focused dark theme
- Icons: Lucide React
- Notifications: React Hot Toast
